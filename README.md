# KKTerminalQT
QT version of KKTerminal  
  
This has replaced KKTerminal.  
```code

```
To install use:
```code
autogen.sh --prefix=/usr
make
make install

```
To install using QT5 instead of the default QT6:
```code
autogen.sh --prefix=/usr --enable-qt5
make
make install

```

There is an AppImage in releases supplied 'as is' tested on mint and void.
